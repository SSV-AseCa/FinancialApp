# Workflow: Investor's Portfolio Performance Measurement
- **Actor:** Investor
- **Goal:** Measure portfolio performance over time
## Steps
1. Investor accesses its portfolio
2. Investor requests portfolio performance metrics
3. System retrieves the latest stored market prices for all portfolio positions
4. System calculates:
	- current portfolio value
	- profits/losses for each position
5. System presents the calculated metrics
6. Investor evaluates the received metrics
