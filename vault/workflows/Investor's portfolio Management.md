# Workflow: Investor's Portfolio Management
- **Actor:** Investor
- **Goal:** Build and maintain a stock portfolio
## Steps
1. Investor access  their portfolio for the first time
2. System creates an empty portfolio for the investor
3. Investor adds stocks to portfolio by specifying:
	- ticker
	- quantity
	- operation date
4. System stores the position and updates the portfolio
5. Investor reviews the updated portfolio
6. Investor adds additional positions if needed
7. Investor modifies or removes positions if needed
8. System persists all portfolio changes
