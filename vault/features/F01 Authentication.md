# Authentication

## Description
Covers investor account creation, authentication, and session management. Every investor must be authenticated before accessing any protected feature. Portfolio provisioning is automatic on registration.

## Goals
- Allow investors to register and access the platform
- Authenticate returning investors securely
- Allow investors to terminate their session
- Provision an empty portfolio automatically on registration
