# Unit Tests

## User Story
As a developer, I want unit tests covering core domain logic so that regressions in isolated components are detected early.

## Acceptance Criteria
- Unit tests are implemented using xUnit or Specs as required
- Core domain logic is covered by unit tests
- Tests run as part of the CI/CD pipeline
