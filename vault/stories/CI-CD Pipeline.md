# CI-CD Pipeline

## User Story

As a developer, I want an automated CI/CD pipeline so
that builds and tests run on every push without manual intervention.

## Acceptance Criteria

- The pipeline is configured in the SaaS platform (GitHub/GitLab/Bitbucket)
- The pipeline triggers on every push
- The pipeline executes automated builds and test suites
- Pipeline failures are reported and block merges
