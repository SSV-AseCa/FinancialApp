# Search Companies

## User Story
As an investor, I want to search for companies by name or ticker so that I can find a company to evaluate.

## Acceptance Criteria
- The investor enters a name or ticker as the search query
- The system queries SEC EDGAR and returns matching companies
- Results are presented to the investor for selection
- The search flow is verified end-to-end on web via Cypress
- The search flow is verified end-to-end on mobile via Appium
