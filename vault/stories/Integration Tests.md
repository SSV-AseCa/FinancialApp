# Integration Tests

## User Story
As a developer, I want integration tests that exercise real external APIs so that integration points are verified against actual behaviour, not mocks.

## Acceptance Criteria
- Integration tests call real external APIs (EDGAR, Yahoo Finance)
- Tests are implemented using xUnit or Specs as required
- Tests run as part of the CI/CD pipeline
