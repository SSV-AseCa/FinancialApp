# Register Account

## User Story
As an investor, I want to register an account so that I can access the platform.

## Acceptance Criteria
- The investor completes the registration flow through Auth0
- Auth0 creates the investor's credentials and account
- The investor is redirected back to the application after successful registration
- The registration flow is verified end-to-end on web via Cypress
- The registration flow is verified end-to-end on mobile via Appium
