# Register Account

## User Story
As an investor, I want to register an account so that I can access the platform.

## Acceptance Criteria
- The investor provides the required registration data
- The system creates the investor's account
- The system automatically creates an empty portfolio for the new investor
- The investor is informed of successful registration
