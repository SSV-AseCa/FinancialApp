# Modify Position — API

## Description

Implement `PUT /portfolio/positions/{positionId}` endpoint that updates an existing position. Accepts the same fields as Add Position (ticker, quantity, operation date) — all fields are required.

## Notes

- Returns 401 if not authenticated
- Returns 404 if the position does not exist or belongs to another investor
- Returns 400 for invalid field values (same rules as T02.02 API)
- Returns 200 with the updated position on success
- The investor can only modify their own positions — never another investor's
