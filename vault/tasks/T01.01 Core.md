# Register Account — Frontend Core

## Description

Integrate the Auth0 SDK and expose a shared `register()` action that initiates the Auth0 registration flow and handles the OAuth callback. Consumed by both web and mobile screens.

## Notes

- Configure the Auth0 SDK with the tenant domain and client ID (provided by T01.Auth0 API)
- After the callback, store the received access token securely for use in subsequent requests
- Coordinate the action interface with Web and Mobile early so all three can work in parallel
- Keep the action decoupled from platform-specific navigation
