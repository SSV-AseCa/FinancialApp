# View Transaction History — Mobile

## Description

Build the mobile transaction history screen that lists all transactions using the `fetchTransactionHistory()` action from `ui-core`, ordered newest-first.

## Notes

- Each row displays: transaction type (buy/sell), company, quantity, and date
- Show a clear empty-state message if the investor has no transactions
- Show a loading indicator while the list is being fetched
- Depends on T04.04 Core
