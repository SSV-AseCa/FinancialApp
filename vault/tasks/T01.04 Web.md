# Log Out — Web

## Description

Expose the logout action on the web UI (e.g. button in the navigation header). On logout, invoke the shared core logout action which clears the token and redirects to Auth0's logout endpoint.

## Notes

- Agree the Core logout action interface upfront; implementation can proceed in parallel
- After logout, the investor must not be able to navigate back to protected screens without re-authenticating
- The post-logout landing page must be a public (unauthenticated) screen
