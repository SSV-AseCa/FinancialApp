# Sell Shares — Cypress

## Description

Write Cypress E2E tests for the web sell-shares flow.

## Scenarios

- Happy path: investor sells a quantity they hold and the portfolio position decreases accordingly
- Oversell: investor attempts to sell more shares than they hold and receives a clear error
- Close position: investor sells all shares of a position and the position is removed from the portfolio

## Notes

- Use `cy.loginByAuth0Api()` for authentication
- Seed a buy transaction via API call in test setup to establish a known position
- Depends on T04.03 Web being complete
