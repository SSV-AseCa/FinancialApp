# Search Companies — Cypress

## Description

Write Cypress E2E tests for the web company search flow.

## Scenarios

- Happy path: investor enters a valid query and receives a list of matching companies
- No results: entering a query with no matches shows the empty-state message
- Blank query: submitting an empty search shows a validation error or does not trigger the search

## Notes

- Use `cy.loginByAuth0Api()` for authentication
- The search calls the real EDGAR API — use a stable test ticker/name (e.g., "AAPL" or "Apple") to avoid flakiness
- Depends on T06.01 Web being complete
