# View Portfolio — Frontend Core

## Description

Expose a `fetchPortfolio()` action in `ui-core` that calls `GET /portfolio` and returns the portfolio data. Provide the portfolio state and loading/error status to consuming screens.

## Notes

- Include the `Authorization: Bearer {token}` header using the token stored during login (T01.03 Core)
- Define and export the `Portfolio` and `Position` types from `src/index.ts`
- Coordinate the interface with Web and Mobile so both screens can consume it without change
- Depends on T01.03 Core for the stored access token
