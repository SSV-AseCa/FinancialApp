# Search Companies — API

## Description

Implement `GET /companies/search?q={query}` endpoint that queries SEC EDGAR's full-text search API and returns matching companies (name and ticker/CIK). All EDGAR requests must pass through the rate-limiting layer (T03.01 API).

## Notes

- Returns 401 if not authenticated
- Returns 400 if the `q` parameter is missing or blank
- The EDGAR rate limiter (T03.01 API) must be in the call path — callers must not bypass it
- Response: list of companies, each with at minimum: company name and CIK
- Cache results for repeated identical queries to reduce EDGAR calls (strategy TBD — document the choice)
- Prerequisite for T03.03 API (Fetch and Store Company Financial Data)
