# Search Companies — Mobile

## Description

Build the mobile company search screen with a search input that calls `searchCompanies()` from `ui-core` and displays the results.

## Notes

- Debounce or gate the search call to avoid firing on every keystroke
- Show a loading indicator while the search is in progress
- Display a clear empty-state message when no companies match the query
- Tapping a result navigates to the company detail screen (to be built in cycle 4)
- Depends on T06.01 Core
