# View Portfolio — Cypress

## Description

Write Cypress E2E tests for the web portfolio view. Authenticate programmatically via Auth0 to avoid cross-origin UI interaction.

## Scenarios

- Happy path: authenticated investor navigates to portfolio screen and sees their positions
- Empty portfolio: authenticated investor with no positions sees the empty-state message
- Unauthenticated access: investor without a token is redirected to the login screen

## Notes

- Use `cy.loginByAuth0Api()` to authenticate programmatically (same approach as T01.01 Cypress)
- Seed portfolio positions via API calls in the test setup using an authenticated token, not through the UI
- Depends on T02.01 Web being complete
