# Log Out — Frontend Core

## Description

Implement the shared `logout()` action: clear the locally stored access token, reset authenticated state, and redirect to Auth0's logout endpoint to terminate the Auth0 session.

## Notes

- Auth0 logout URL: `https://your-tenant.auth0.com/v2/logout?returnTo=<app-url>&client_id=<client-id>`
- The local token must be cleared before initiating the Auth0 redirect
- After logout, the shared auth state must be fully cleared before navigation
- Keep the action decoupled from platform-specific navigation
