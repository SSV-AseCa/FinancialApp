# Auto-Create Portfolio on Registration — API

## Description

Within the registration transaction, create an empty portfolio and associate it exclusively with the new investor. This is not a separate endpoint — it is part of the registration operation.

## Notes

- Enforces the invariant: exactly one portfolio per investor
- Must be atomic with investor creation — no investor without a portfolio, no orphan portfolios
- No separate call from the client; triggered entirely server-side
