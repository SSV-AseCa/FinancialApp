# View Portfolio — Web

## Description

Build the web portfolio screen that displays the authenticated investor's current positions using the `fetchPortfolio()` action from `ui-core`.

## Notes

- Redirect to login if the investor is not authenticated
- Show a loading indicator while the portfolio is being fetched
- Display each position with: ticker, quantity, and operation date
- Show a clear empty-state message if the portfolio has no positions
- Depends on T02.01 Core
