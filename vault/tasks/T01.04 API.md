# Log Out — API

## Description

Implement the logout endpoint. Invalidate the investor's current session or token so that subsequent requests with it are rejected.

## Notes

- Endpoint: POST /auth/logout
- Strategy must be consistent with the token approach defined in T01.03.API
- After logout, any request using the invalidated token must return 401
