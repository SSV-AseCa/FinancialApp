# Search Companies — Frontend Core

## Description

Expose a `searchCompanies(query)` action in `ui-core` that calls `GET /companies/search?q={query}` and returns the list of matching companies.

## Notes

- Include the `Authorization: Bearer {token}` header
- Export the `Company` type (name, CIK) from `src/index.ts`
- Handle empty results as an empty list, not an error
- Coordinate the interface with Web and Mobile so both can work in parallel
- Depends on T01.03 Core for the stored access token
