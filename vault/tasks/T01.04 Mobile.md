# Log Out — Mobile

## Description

Expose the logout action on the mobile UI. On logout, invoke the shared core logout action which clears the token and terminates the Auth0 session.

## Notes

- Agree the Core logout action interface upfront; implementation can proceed in parallel
- After logout, the investor must not be able to navigate back to protected screens without re-authenticating
- Use the Auth0 SDK's logout method to correctly terminate the native session
