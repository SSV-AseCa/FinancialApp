# Fetch and Store Market Prices — API

## Description

Implement a scheduled background job that fetches current market prices from Yahoo Finance for all tickers present in any investor's portfolio and persists them. Persisted prices become the sole source of truth for portfolio analytics — live calls at query time are not permitted.

## Notes

- Define and document the fetch frequency (e.g., every 15 minutes during market hours, once daily after close) — justify the choice in a comment or doc
- Store each fetched price with the ticker, price value, and timestamp
- Only fetch tickers that exist in at least one portfolio — avoid unnecessary Yahoo Finance calls
- Yahoo Finance API key is sourced from the `YAHOO_FINANCE_API_KEY` environment variable
- This task is a prerequisite for S05.01 (View Portfolio Total Value) and S05.02 (View Position Profit and Loss) in cycle 4
- The EDGAR rate limiter (T03.01) does NOT apply here — Yahoo Finance has its own rate-limit characteristics; document and handle them
