# Add Position — Frontend Core

## Description

Expose an `addPosition(ticker, quantity, operationDate)` action in `ui-core` that calls `POST /portfolio/positions` and returns the created position or a validation error.

## Notes

- Include the `Authorization: Bearer {token}` header
- Surface validation error messages from the API response to the caller
- Export the `AddPositionInput` input type from `src/index.ts`
- Coordinate the action interface with Web and Mobile so both can work in parallel
- Depends on T02.01 Core for the stored access token
