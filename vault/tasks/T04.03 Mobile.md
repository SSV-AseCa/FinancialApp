# Sell Shares — Mobile

## Description

Build the mobile sell-shares form that submits a sell operation via the `sellShares()` action from `ui-core`.

## Notes

- Form fields: company (pre-selectable from portfolio positions), quantity
- Display validation and business-rule errors — especially the insufficient-shares case — with a user-friendly message
- On success: reflect the updated portfolio positions (re-fetch or optimistically update)
- Depends on T04.03 Core and T02.01 Mobile
