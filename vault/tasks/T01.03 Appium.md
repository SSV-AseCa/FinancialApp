# Log In — Appium

## Description

Write the Appium E2E test for the mobile login flow. Tests obtain a token directly from Auth0 to avoid driving Auth0's UI.

## Scenarios

- Happy path: programmatic Auth0 login → access token obtained → investor reaches portfolio screen
- Unauthenticated access: attempt to navigate to a protected screen → redirected to login
- Post-login access: authenticated investor can reach protected screens

## Notes
