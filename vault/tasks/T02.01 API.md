# View Portfolio — API

## Description

Implement `GET /portfolio` endpoint that returns the authenticated investor's portfolio and all current positions. The investor is identified by the `sub` claim injected by the JWT middleware (T01.Auth0 API) and provisioned by the lazy provisioning middleware (T01.02 API).

## Notes

- Returns 401 if the request is not authenticated
- An investor must never see another investor's portfolio — scope the query to the authenticated investor's ID
- Response includes: portfolio ID, list of positions (each with: position ID, ticker, quantity, operation date)
- Empty portfolio returns an empty positions list, not a 404
- Depends on T01.02 API (provisioning) being in place
