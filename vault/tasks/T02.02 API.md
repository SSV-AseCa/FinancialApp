# Add Position — API

## Description

Implement `POST /portfolio/positions` endpoint that adds a new position to the authenticated investor's portfolio. Accepts: ticker (string), quantity (positive integer), operation date (ISO 8601 date).

## Notes

- Returns 401 if not authenticated
- Returns 400 with a descriptive error body if any field is missing, the ticker is blank, quantity is not a positive integer, or the date is invalid
- Returns 201 with the created position on success
- Scope the operation to the authenticated investor's portfolio — an investor cannot add positions to another investor's portfolio
- Depends on T02.01 API (portfolio endpoint) being in place
