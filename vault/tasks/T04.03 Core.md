# Sell Shares — Frontend Core

## Description

Expose a `sellShares(companyCik, quantity)` action in `ui-core` that calls `POST /portfolio/transactions/sell` and returns the created transaction or a validation/business error.

## Notes

- Include the `Authorization: Bearer {token}` header
- Surface 400 and 422 error messages — especially the insufficient-shares error — to the caller
- Export the `SellSharesInput` type from `src/index.ts`
- Coordinate the interface with Web and Mobile so both can work in parallel
- Depends on T04.02 Core (shared Transaction type)
