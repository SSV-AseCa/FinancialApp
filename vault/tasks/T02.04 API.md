# Remove Position — API

## Description

Implement `DELETE /portfolio/positions/{positionId}` endpoint that removes an existing position from the authenticated investor's portfolio.

## Notes

- Returns 401 if not authenticated
- Returns 404 if the position does not exist or belongs to another investor
- Returns 204 No Content on successful removal
- An investor can only remove their own positions — never another investor's
- Depends on T02.01 API
