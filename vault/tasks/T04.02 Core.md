# Buy Shares — Frontend Core

## Description

Expose a `buyShares(companyCik, quantity)` action in `ui-core` that calls `POST /portfolio/transactions/buy` and returns the created transaction or a validation/business error.

## Notes

- Include the `Authorization: Bearer {token}` header
- Surface 400 and 422 error messages from the API response to the caller
- Export the `BuySharesInput` and `Transaction` types from `src/index.ts`
- Coordinate the interface with Web and Mobile so both can work in parallel
- Depends on T01.03 Core for the stored access token
