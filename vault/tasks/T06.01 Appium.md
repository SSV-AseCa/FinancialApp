# Search Companies — Appium

## Description

Write Appium E2E tests for the mobile company search flow.

## Scenarios

- Happy path: investor enters a valid query and receives a list of matching companies
- No results: entering a query with no matches shows the empty-state message
- Blank query: submitting an empty search shows a validation error or does not trigger the search

## Notes

- Obtain a test token directly from Auth0's token endpoint
- Use a stable test ticker/name (e.g., "AAPL" or "Apple") to keep the test deterministic
- Depends on T06.01 Mobile being complete
