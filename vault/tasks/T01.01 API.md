# Register Account — API

## Description

Implement the registration endpoint. Validate all required input fields, hash the password, and persist the new investor.

## Notes

- Endpoint: POST /auth/register
- Required fields to define with the team before frontend starts
- Password must be hashed before persistence — never stored in plain text
- Registration and portfolio creation must be a single transaction; if portfolio creation fails, the investor must not be persisted
